## Commands 













#### Blockchain part
Set-up
```shell
- cd blockchain 
- npm init -y
- npm install --save-dev @nomicfoundation/hardhat-toolbox
- npx hardhat
- npm i @openzeppelin/contracts
```

After writing code:
```
- npx hardhat compile
- npx hardhat run --network sepolia scripts/deploy.js
```
